# 🛡️ Cyber Threat Intelligence Toolkit

**Author:** MMAKOLA THATO  
**Goal:** Distinction-level cybersecurity portfolio project demonstrating Python, JSON, CTI, STIX 2.1, IOC handling, malware analysis concepts, and clean software engineering practices.

---

## 🚀 Features

- 🔍 IOC Validator (hashes, IPs, domains, URLs, emails)
- 📦 STIX 2.1 Bundle Generator with MITRE ATT&CK mapping
- 🦠 Malware Static Analysis Assistant (hashes, entropy, risk scoring)
- 🎨 Colored CLI Dashboard with Rich
- 📊 Export results to JSON or CSV
- 🌐 Pull latest MITRE ATT&CK data automatically
- 📝 IOC validation reports
- 📈 Summary statistics
- 🛡️ Detection recommendations mapped to each technique
